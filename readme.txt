### Winsome Hover Gallery by http://abdullahmasum.elance.com/

This plugin will enable nice hover effects with portfolio and image gallery on your wordpress site .

Plugin Features

* Unlimited image gallery.
* Extra-ordinary hover effects
* Strong customization supports
* Mobile touch option. 
* Very Lightweight.
* Works on every modern browser.
* Gallery controll



== Installation ==
<pre>
* Install it as a regular WordPress plugin
* After installing this plugin you can use these shortcodes

   1.[hover_gallery]
   2.[hover_gallery category="your-category-name"]

</pre>
